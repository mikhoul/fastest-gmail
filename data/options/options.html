Popup Size Toolbar Popup width is px.

Minimum popup width is 330px and default value is 600px.

Toolbar Popup height is px.

Minimum popup height is 400px and default value is 500px.
Badge Icon Text Check Gmail Inbox every seconds.

Minimum time interval is 60 seconds. Default is 300 seconds.
Gmail Page Open Gmail in (inbox.google.com)

Check this box if you want to open Gmail in (inbox.google.com) when clicking on the Gmail button in toolbar-popup.
Gmail Labels Enter the name(s) of your Gmail Label(s) that you want to track (for the primary account)

Labels are comma (,) separated. i.e. label1, label2, label3, ...
Desktop Notifications Check this box if you want to show desktop notification popup for new emails

The popup will close automatically after few seconds.
Number of Notifications Enter the desired maximum number of desktop notifications you want to see

The minimum number is 1. If you want to disable Desktop Notifications, see the above option.
Beep Sound Check this box if you want to hear a beep sound when a new email arrives

The beep sound lasts 1 second.
Badge Background Color Pick a color for badge-icon background

Default badge background color is red.
